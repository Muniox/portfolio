01O mnie

Tworzę nowoczesne aplikacje webowe łączące elegancki frontend z solidnym backendem

Jestem full-stack developerem specjalizującym się w ekosystemie Angular oraz platformie .NET. Od ponad 5 lat projektuję i wdrażam skalowalne aplikacje webowe — od interaktywnych dashboardów po złożone systemy enterprise.

Moją pasją jest czysty kod, reaktywne wzorce projektowe i tworzenie interfejsów, które są zarówno piękne, jak i wydajne.

Angular 17+ TypeScript .NET 8 C# RxJS SQL SignalR
about.service.ts 
@Injectable({ providedIn: 'root' })
export class AboutService {

  readonly passion = 'clean code';

  readonly stack: Tech[] = [
    { name: 'Angular', level: 'expert' },
    { name: '.NET',    level: 'advanced' },
    { name: 'RxJS',   level: 'advanced' },
  ];

  buildApp(): Observable<App> {
    return of(this.stack).pipe(
      map(s => compose(s)),
      tap(() => console.log('shipped ✓'))
    );
  }
}
